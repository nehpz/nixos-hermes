# ADR-004: No Swap

**Status:** Accepted

## Context

Swap space can be used to extend effective RAM at the cost of I/O latency. On NVMe drives, swap I/O is fast but still orders of magnitude slower than RAM.

## Decision

There is no swap partition or swap file. `vm.swappiness = 0` prevents the kernel from speculatively moving pages to swap. This is justified by 96GB DDR5 RAM, which exceeds the expected working set of `hermes-agent` and its toolsets.

## Consequences

If memory pressure exceeds 96GB, the OOM killer activates rather than degrading to swap. Revisit this decision if memory-intensive workloads are added.
