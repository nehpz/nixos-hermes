# ADR-002: Firewall Disabled

**Status:** Accepted

## Context

The host sits on a trusted LAN segment behind a gateway firewall. An additional host-level firewall adds operational friction, with port rules to maintain for each service added, while providing minimal security benefit on this network segment.

## Decision

`networking.firewall.enable = false`. The network perimeter is the trust boundary.

## Consequences

All ports the host listens on are reachable from the LAN. Any service that binds to `0.0.0.0` is accessible network-wide. Revisit this decision if the host ever moves to an untrusted network segment.
